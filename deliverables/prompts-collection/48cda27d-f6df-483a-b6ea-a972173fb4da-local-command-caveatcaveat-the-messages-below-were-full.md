Yes, there are several options for getting Brazilian stock market (B3) data for free, though "real-time" comes with caveats:

## Truly Free Options

| Source | Delay | Notes |
|--------|-------|-------|
| **Yahoo Finance API** (`yfinance` Python lib) | ~15 min | Append `.SA` to tickers (e.g., `PETR4.SA`). Most popular free option. |
| **Google Finance** | ~15 min | Available via Google Sheets `=GOOGLEFINANCE("BVMF:PETR4")` |
| **Brapi** (brapi.dev) | Near real-time | Brazilian API, free tier with rate limits. REST + WebSocket. |
| **Alpha Vantage** | 15 min | Free tier: 25 requests/day. Supports B3 tickers. |
| **HG Finance API** | Varies | Brazilian API with stock quotes. |
| **StatusInvest / Fundamentus** | Varies | Can be scraped (check ToS). Good for fundamentals. |

## Near Real-Time / Free Tiers

- **Brapi** is probably the best Brazilian-specific option — it has a free tier with near real-time quotes and is designed for B3 data.
- **TradingView WebSocket** — unofficial, but people reverse-engineer their WebSocket for real-time data (fragile, may break).

## Important Caveats

1. **True real-time B3 data requires a paid subscription** from B3 itself or authorized distributors (e.g., Cedro Technologies, Enfoque). The exchange charges for real-time feeds.
2. Most free sources have a **15-minute delay** — this is standard for exchanges worldwide.
3. **Rate limits** apply to all free tiers.

## Quick Example with `yfinance`

```python
import yfinance as yf

# Fetch Petrobras (PETR4) data
ticker = yf.Ticker("PETR4.SA")
print(ticker.info["currentPrice"])
hist = ticker.history(period="1d", interval="1m")
```

## Quick Example with Brapi

```python
import requests

resp = requests.get("https://brapi.dev/api/quote/PETR4")
data = resp.json()
print(data["results"][0]["regularMarketPrice"])
```

**Bottom line:** For hobby/personal projects, **Brapi** or **yfinance** are your best bets. For actual real-time trading, you'll need a paid feed.

Yes, several of those free options provide fundamental data beyond just price quotes.

## Best Options for Fundamentals

### yfinance — Most Complete

```python
import yfinance as yf

ticker = yf.Ticker("PETR4.SA")
info = ticker.info

# Valuation
info["trailingPE"]          # P/E (Preço/Lucro)
info["forwardPE"]           # P/E forward
info["priceToBook"]         # P/VP (Preço/Valor Patrimonial)
info["bookValue"]           # VPA (Valor Patrimonial por Ação)
info["enterpriseValue"]     # EV

# Profitability
info["returnOnEquity"]      # ROE
info["profitMargins"]       # Margem líquida
info["operatingMargins"]    # Margem operacional

# Dividends
info["dividendYield"]       # Dividend Yield
info["trailingAnnualDividendRate"]

# Financials (full statements)
ticker.financials           # DRE (Income Statement)
ticker.balance_sheet        # Balanço Patrimonial
ticker.cashflow             # Fluxo de Caixa
```

**Caveat:** yfinance data for `.SA` tickers can sometimes be incomplete or slightly outdated for some fields.

### Brapi — Good for Brazilian-Specific Data

```python
import requests

resp = requests.get("https://brapi.dev/api/quote/PETR4", 
                     params={"fundamental": "true"})
data = resp.json()["results"][0]

# earningsPerShare, priceEarnings, bookValue, etc.
```

### StatusInvest — Richest Fundamental Data for B3

Not an official API, but their pages expose structured data:

```python
import requests

headers = {"User-Agent": "Mozilla/5.0"}
resp = requests.get("https://statusinvest.com.br/acoes/petr4", headers=headers)
# Parse HTML or use their internal JSON endpoints
```

StatusInvest has the most complete set of Brazilian fundamentals (P/L, P/VP, ROE, ROIC, Dívida Líquida/EBITDA, DY, Payout, etc.) — similar to what you'd see on Fundamentus.

## Comparison

| Source | P/L | P/VP | ROE | Balance Sheet | DRE | Reliability for B3 |
|--------|-----|------|-----|---------------|-----|---------------------|
| **yfinance** | Yes | Yes | Yes | Yes | Yes | Good, some gaps |
| **Brapi** | Yes | Yes | Partial | No | No | Good |
| **StatusInvest** | Yes | Yes | Yes | Yes | Yes | Best (B3-focused) |
| **Alpha Vantage** | Yes | Yes | Yes | Yes | Yes | Inconsistent for .SA |

**Bottom line:** For a project that needs P/L, P/VP, and balance sheet data for Brazilian stocks, **yfinance** is the easiest to code against, and **StatusInvest** is the most reliable source (though you'd need to scrape it).

```python
import yfinance as yf

# Some major B3 tickers
tickers = [
    "PETR4.SA", "VALE3.SA", "ITUB4.SA", "BBDC4.SA", "ABEV3.SA",
    "WEGE3.SA", "RENT3.SA", "BBAS3.SA", "SUZB3.SA", "GGBR4.SA",
    "CSNA3.SA", "RADL3.SA", "JBSS3.SA", "CMIG4.SA", "TAEE11.SA",
]

results = []
for t in tickers:
    try:
        pe = yf.Ticker(t).info.get("trailingPE")
        if pe and pe > 0:  # skip negative/missing
            results.append((t.replace(".SA", ""), pe))
    except Exception:
        continue

results.sort(key=lambda x: x[1])

print("Top 5 — Lowest P/L")
for ticker, pe in results[:5]:
    print(f"  {ticker:<8} P/L: {pe:.2f}")
```

**Note:** This is slow because `yfinance` makes one HTTP call per ticker. To scan the full B3, you'd want to either:
- Use `yf.download(tickers, period="1d")` for batch price data (faster, but no fundamentals)
- Scrape StatusInvest/Fundamentus which already have pre-computed rankings
- Use Brapi's batch endpoint

There's no single free API call that gives you the full B3 ticker list, but here are practical approaches:

## 1. Scrape B3 Directly

B3 publishes a daily file with all listed instruments:

```python
import requests
import pandas as pd
from io import StringIO

# B3's public listing page (JSON endpoint)
url = "https://sistemaswebb3-listados.b3.com.br/listedCompaniesProxy/CompanyCall/GetInitialCompanies/"
payload = '{"language":"pt-br","pageNumber":1,"pageSize":120}'

headers = {"Content-Type": "application/json"}
resp = requests.post(url, data=payload, headers=headers)
data = resp.json()

for company in data["results"]:
    print(company["issuingCompany"], company["companyName"])
```

**Caveat:** Paginated, you need to loop through pages.

## 2. Use the IBOV Composition (Simpler)

If you only care about the ~80–90 most liquid stocks:

```python
import pandas as pd

# B3 publishes IBOV composition as CSV
url = "https://raw.githubusercontent.com/riconnon/acao_ibovespa/main/ibov.csv"
# Or scrape from B3's theoretical portfolio page
```

## 3. Pragmatic Approach — Curated List from StatusInvest

from bs4 import BeautifulSoup

# StatusInvest lists all ~400+ traded tickers on this page

## 4. CVM Open Data (Official)

CVM (Brazilian SEC) publishes open datasets:

```python
# All registered companies with tickers
url = "https://dados.cvm.gov.br/dados/CIA_ABERTA/CAD/DADOS/cad_cia_aberta.csv"
df = pd.read_csv(url, sep=";", encoding="latin1")
# Filter for active companies
active = df[df["SIT"] == "ATIVO"]
print(active[["DENOM_SOCIAL", "CD_CVM"]].head())
```

This gives you companies but not trading tickers directly — you'd need to map them.

## Most Practical Combo

For a screening project, I'd combine **CVM data** (official, complete) with **yfinance validation**:

```python
# Quick validation that a ticker exists on yfinance
def is_valid(ticker):
    try:
        info = yf.Ticker(f"{ticker}.SA").info
        return info.get("regularMarketPrice") is not None
    except:
        return False
```

**Bottom line:** There's no clean one-liner. The most reliable path is scraping B3's own listing or using CVM open data, then appending `.SA` for yfinance lookups.
