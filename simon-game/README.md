# Simon Game

A classic Simon memory game built with Next.js, TypeScript, and Tailwind CSS.

## How to Play

1. Press **Start Game** to begin.
2. Watch the sequence of colors light up.
3. Repeat the sequence by clicking the colored buttons in the same order.
4. Each round adds one more color to the sequence.
5. If you press the wrong color, the game is over.

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command         | Description                  |
| --------------- | ---------------------------- |
| `npm run dev`   | Start development server     |
| `npm run build` | Create production build      |
| `npm run start` | Start production server      |
| `npm run lint`  | Run ESLint                   |

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
